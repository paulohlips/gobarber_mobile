## Let's run

```bash
# Clone this repository
$ git clone https://github.com/paulohlips/gobarber_mobile

# Go into the repository
$ cd gobarber_mobile

# Install dependencies
$ yarn install

# Run the app
$ react-native run-android ( or run-ios)
```
